Flexbox for dummies

What is flexbox? Everything on a page is a box. Boxes containing boxes containing boxes... Flexbox is a way to arrange them Normally that is called layout model.

Basics

Basics Let's say we want to arrange our site horizontally. Create a box containing 3 more boxes inside of it. We will use <div> to implement this. *interactive code of 1 div containing 3 more divs inside, all colored differently* To start with flexbox we have to specify display: flex as a css property for outer box, that is our flex container. From now on every single element contained inside of it is referred to as flex item. *same interactive code but with flexbox properties on container and labels “item” on smaller boxes* So, box that contains - container. Boxes that are contained - items. Let's move on.

As you have noticed it arranged our items in a straight line. That line is called main axis. It can either be horizontal(row) spanning left to right, or vertical(column) spanning top to bottom. *static picture of items arranged in row, with arrow in that direction titled “main”* *same but with column* There is also a second line, called cross axis, which will always be perpendicular to the main axis. *static picture of intersected main and cross axis*

Container properties

flex-direction:

Self-explanatory this sets the direction for the main line. Try changing the values to see the effect. Possible values are: row, column, reverse-row, reverse-column.

justify-content:

This specifies how we align items along the main axis. Go play with it, possible values are: center, flex-start, flex-end, space-between, space-around, space-evenly.

align-items:

And in this we specify how to align items along cross axis. Again, go and try it for yourself. Values are: flex-start, flex-end, center, baseline, stretch. You may noticed have noticed that the default behavior is stretch.

flex-wrap:

This specifies wether we want to wrap our content in several lines in case of overflow(e.g. when our container becomes so small that it's items start to peak out, this occurs when, for example, screen width is too narrow) Values are: wrap, wrap-reverse

Flex items properties

flex-grow:

specifies how much item will grow relative to other items. Grow is only when we have a free space along main axis. I could bloat you with in depth explanation of how this works. But in practice, you will mostly need two values for this property: 0 - it won't grow, 1 - it will grow. Default is 0.

Flex-shrink:

much as the flex-grow, it determines how an item will shrink. Similarly, 0 - it won't shrink, 1 - it will.

Align-self:

basically, the same as align-items property of container. But it affects only 1 child instead of all of them.
