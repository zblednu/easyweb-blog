Must-have VSCode Extension

Open up your VSCode and install this:

What it does is it will allow you to see changes made to the page you're working on in real-time. Much better than spamming the "F5" button.
Just open any html file with VSCode, press this icon in the top-right corner and try it out:

! Be aware that the changes will be applied immidiately as you make them, without the need for you to save the file (extension creates copy of your working folder that is auto-saved on any change). So keep in mind that some stupid thing like accidental infinite loop in javascript will be executed as soon as the closing bracket for it is typed. And the next thing you see is "VSCode not responding". Yes, that happened.
