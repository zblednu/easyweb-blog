Flexbox for dummies

What is flexbox? Everything on a page is a box. Boxes containing boxes containing boxes... Flexbox is a way to arrange them Normally that is called layout model.

Basics

To start with flexbox we have to specify display: flex as css property for the outer box, that is our flex container. From now on every single element contained inside of it is referred to as flex item.
Try adding display: flex to container properties. So, box that contains - container.
Boxes that are contained - items.
Let's move on.

Container properties

flex-direction:

As you have noticed it arranged our items in a straight line. That line is called main axis. It can either be horizontal(row) spanning left to right, or vertical(column) spanning top to bottom. flex-direction sets the direction for the main axis. Try changing the values to see the effect.
Possible values are: row, column, reverse-row, reverse-column.

justify-content:

This specifies how we align items along the main axis. Go play with it, possible values are: center, flex-start, flex-end, space-between, space-around, space-evenly.

align-items:

And in this we specify how to align items along cross axis. Again, go and try it for yourself. Values are: flex-start, flex-end, center, baseline, stretch.

You may noticed have noticed that the default behavior is stretch.

flex-wrap:

This specifies wether we want to wrap our content in several lines in case of overflow(e.g. when our container becomes so small that it's items start to peak out, this occurs when, for example, screen width is too narrow) Values are: wrap, wrap-reverse

Item properties

flex-grow:

Specifies how much item will grow relative to other items. Grow is only when we have a free space along main axis. I could bloat you with in depth explanation of how this works. But in practice, you will mostly need two values for this property: 0 - it won't grow, 1 - it will grow. Default is 0.

flex-shrink:

much as the flex-grow, it determines how an item will shrink. Similarly, 0 - it won't shrink, 1 - it will.
