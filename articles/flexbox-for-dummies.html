Flexbox for dummies

What is flexbox?
Everything on a page is a box. Boxes containing boxes containing boxes...
Flexbox is a way to arrange them. Normally that is called layout model.

Basics

To start with flexbox we have to specify display: flex as a css property for the outer box, that is the flex container. From now on every single element contained inside is referred to as flex item.
Add display: flex to container properties: So, the box that contains - container.
Boxes that are contained - items.
Let's move on.

Container properties

flex-direction

As you have noticed it arranged our items in a straight line. That line is called main axis. It can either be horizontal(row) spanning left to right, or vertical(column) spanning top to bottom.
flex-direction sets the direction for main axis. Try changing it to see the effect.
Possible values are: row, column, row-reverse, column-reverse

justify-content

This one aligns items along the main axis.
Go play with it, possible values are: center, flex-start, flex-end, space-between, space-evenly

align-items

And this one aligns along the cross axis(the one perpendicular to main axis).
Again, go and try it for yourself.
Values: flex-start, flex-end, center, stretch

Note that the default behavior is stretch.

flex-wrap

This specifies wether to wrap items in several lines in case of overflow (e.g. when our container becomes so small that it's items start to peak out, this occurs when, for example, screen width is too narrow) Values: wrap, wrap-reverse

Item properties

! This is where i trimmed the most info. I've left only what is used on the day-to-day basis. Be aware that it's not everything there is, and you might need to dig deeper when working on more complicated projects.

flex-grow

Specifies wether the item should grow along main axis to fill empty space.
0 - it won't grow
1 - it will

Note that the default value is 0.

flex-shrink

Same as the flex-grow, it determines wether an item should shrink to fit.
Similarly,
0 - it won't shrink
1 - it will

Note that by default shrink is set to 1!
